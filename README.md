# rust-blockchain-core-suite
Rust 构建的高性能、模块化区块链核心开发套件，集成共识算法、加密安全、交易处理、P2P网络、智能合约、跨链交互、数据存储等全栈区块链功能模块，适用于公链、联盟链、Layer2扩容、去中心化应用等多场景开发，具备低 gas、高并发、强安全、易扩展的特性，支持多语言混合开发与跨平台部署。

## 包含功能文件列表
1. basic_blockchain.rs：基础区块链链式结构、区块生成、哈希校验核心实现
2. blockchain_crypto.rs：区块链专用非对称加密、签名验签、哈希算法工具集
3. merkle_tree.rs：Merkle树数据结构，用于交易数据快速校验与默克尔证明生成
4. p2p_transport.rs：区块链P2P网络传输层，实现节点发现、数据广播、点对点通信
5. consensus_pow.rs：工作量证明（PoW）共识算法完整实现，支持难度动态调整
6. consensus_pos.rs：权益证明（PoS）共识算法，实现质押、出块、惩罚机制
7. transaction_core.rs：区块链交易结构体、签名、验签、UTXO管理核心
8. wallet_core.rs：去中心化钱包，支持密钥生成、地址管理、交易签名发起
9. state_store.rs：区块链世界状态存储，支持键值对读写、快照、回滚功能
10. block_validator.rs：区块合法性校验器，验证哈希、签名、交易、时间戳
11. mempool_manager.rs：交易内存池管理，实现交易排序、去重、打包上链
12. peer_manager.rs：P2P节点管理，维护节点列表、心跳检测、黑名单控制
13. smart_vm_core.rs：轻量级区块链虚拟机，支持智能合约字节码解析与执行
14. contract_deploy.rs：智能合约部署、编译、链上存储与调用权限管理
15. cross_chain_bridge.rs：跨链桥核心模块，实现异构链资产转移与数据互通
16. layer2_state.rs：Layer2扩容方案状态管理，支持链下计算与链上验证
17. gas_calculator.rs：交易Gas费计算模型，动态调整费率、防止网络拥堵
18. crypto_ed25519.rs：Ed25519椭圆曲线加密算法，区块链高性能签名实现
19. rpc_server.rs：区块链RPC服务端，提供HTTP/JSON-RPC接口供外部调用
20. rpc_client.rs：RPC客户端，用于节点间、应用与链的远程接口调用
21. block_syncer.rs：区块同步器，实现新节点快速同步全量区块数据
22. network_encrypt.rs：P2P网络传输加密，保障节点间通信数据安全
23. staking_pool.rs：PoS质押池管理，支持用户质押、收益分配、解质押流程
24. governance_core.rs：链上治理模块，实现提案发起、投票、决议执行功能
25. nft_mint_core.rs：NFT铸造、转让、权属校验核心逻辑实现
26. ft_token_core.rs：同质化代币（FT）发行、转账、余额管理标准实现
27. chain_bootstrap.rs：区块链创世区块生成、网络初始化启动模块
28. data_archive.rs：区块链历史数据归档，压缩存储、减少节点存储压力
29. tx_indexer.rs：交易索引器，快速检索交易哈希、地址、区块关联数据
30. validator_set.rs：PoS验证者集合管理，动态更新验证者列表
31. contract_event.rs：智能合约事件监听、触发、链上日志存储模块
32. zero_knowledge_proof.rs：零知识证明核心，实现隐私交易数据验证
33. chain_monitor.rs：区块链运行监控，统计区块高度、TPS、节点在线状态
34. sharding_core.rs：区块链分片核心，实现数据分片、交易并行处理
35. oracle_core.rs：预言机模块，链下数据可信上链、喂价、数据校验
36. multi_signature.rs：多签钱包核心，支持多节点共同签名确认交易
37. tx_replacement.rs：交易替换机制，支持加速交易、取消未上链交易
38. block_reward.rs：区块出块奖励、矿工激励、代币分发逻辑实现
39. peer_discovery.rs：P2P节点主动发现，局域网+公网节点自动组网
40. chain_fork_handle.rs：链分叉处理模块，实现最长链选择、分叉数据回滚

## 项目特性
- 纯Rust高性能开发，内存安全、无GC、并发能力拉满
- 模块化设计，所有功能解耦，可按需引入使用
- 兼容主流区块链协议，支持快速定制化开发
- 完善的加密体系，保障资产与数据安全
- 支持测试网与主网双模式，一键切换部署
- 完整的工具链，降低区块链开发门槛
